# Books Folder

Place your PDF files here. The filenames should match the `pdfUrl` values in `src/data/books.ts`.

## Example

If a book has `pdfUrl: "/books/quran.pdf"`, place the file at:
```
public/books/quran.pdf
```

## Adding New Books

1. Add the PDF file to this folder
2. Add the book entry in `src/data/books.ts`
3. The book will automatically appear in the library
